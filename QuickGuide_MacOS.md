# QuickGuide | MacOS

## Shortcuts

### General

| Command | Description |
| --- | --- |
| `cmd + space` | Opens Spotlight Search |
| `ctrl + up/down arrow` | Zooms out/in open windows/desktops |

### Finder

| Command | Description |
| --- | --- |
| `ctrl + shift + g` | Go to folder path |
| `ctrl + del` | Move to trash |
| `ctrl + shift + del` | Delete permanently |
