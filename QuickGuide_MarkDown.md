## Markdown Syntax

| Syntax | Preview |
| --- | --- |
| `**Bold**` | **Bold** |
| `*Italic*` | *Italic* |
| `~~Strikethrough~~` | ~~Strikethrough~~ |
| ``  `git add 'code.md'``` | `git add 'code.md'` |


## Code

Syntax:

````
```bash
git add 'file.md'
```
````

Output:

```bash
git add 'file.md'
```

## Task Lists

Syntax:

```
- [x] Task 1
- [ ] Task 2
- [ ] Task 3
```

Output:

  - [x] Task 1
  - [ ] Task 2
  - [ ] Task 3
