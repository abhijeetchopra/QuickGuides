# QuickGuide | Vim

##### Install
```bash
sudo apt intall vim
```

##### Run
```bash
vim <filename>
```

##### Settings
```bash
vim ~/.vimrc
```

##### Two Modes
1. Command mode  
2. Insert mode

## Command mode
Press `esc` in insert mode to switch to command mode
```
:q    # quit
:q!   # force quit
:w    # write
:w!   # force write
:wq   # write, then quit
:x    # alias for :wq
```


## Insert mode
Press `i` in command mode to switch to insert mode
